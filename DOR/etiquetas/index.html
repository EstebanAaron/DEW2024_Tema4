Elementos para incrustar contenido

<img>

Se utiliza para mostrar imágenes.

Atributos más comunes:

  • src: URL o ruta de la imagen.
  • alt: Texto alternativo para accesibilidad o cuando la imagen no carga.
  • width y height: Dimensiones de la imagen (píxeles o porcentaje).

<iframe>

Permite incrustar otro documento HTML.

Atributos más comunes:

  • src: URL del contenido incrustado.
  • width y height: Dimensiones del iframe.
  • title: Texto para describir el iframe (accesibilidad).
  • allow: Especifica permisos (e.g., reproducir videos).
  • sandbox: Restringe funcionalidades (e.g., ejecución de scripts).

<embed>

Se usa para incrustar contenido multimedia o archivos externos.

Atributos más comunes:

  • src: URL del contenido a incrustar.
  • type: Tipo MIME del contenido (e.g., application/pdf).
  • width y height: Dimensiones del elemento.

<object> y <param>

El elemento <object> permite incrustar contenido externo, y <param> define parámetros adicionales.

Atributos más comunes:

  • data: URL del contenido a cargar.
  • type: Tipo MIME del contenido.
  • width y height: Dimensiones del elemento.

<video> y <source>

Se usa para reproducir videos.

Atributos más comunes:

  • controls: Muestra controles para reproducir, pausar, etc.
  • autoplay: El video se reproduce automáticamente.
  • loop: El video se repite en bucle.
  • poster: Imagen que se muestra antes de reproducir el video.

<audio>

Se usa para reproducir audio.

Atributos más comunes:

  • controls: Muestra controles para reproducir, pausar, etc.
  • autoplay: El audio se reproduce automáticamente.
  • loop: El audio se repite en bucle.

<track>

Se usa dentro de <video> o <audio> para añadir subtítulos.

Atributos más comunes:

  • src: URL del archivo de subtítulos.
  • kind: Tipo de pista (subtitles, captions, etc.).
  • srclang: Idioma de los subtítulos.
  • label: Nombre descriptivo para los subtítulos.
  • nota: Para que algunos navegadores carguen archivos vtt es necesario a traves de un servidor

<canvas>

Se usa para dibujar gráficos mediante JavaScript.

Atributos más comunes:

  • width y height: Dimensiones del área de dibujo.

<svg>

Se usa para gráficos vectoriales escalables.

Atributos más comunes:

  • width y height: Dimensiones del gráfico.

<map> y <area>

Se usan para definir áreas interactivas en una imagen, como mapas de enlaces o diagramas.

Atributos más comunes:

  • map: Define el mapa de áreas.
  • area:
    • shape: Forma del área (rect, circle, poly).
    • coords: Coordenadas de la región.
    • href: Enlace asociado al área.
    • alt: Texto alternativo.

<math>

Se usa para representar expresiones matemáticas con el estándar MathML.

Atributos más comunes:

  • xmlns: Espacio de nombres MathML.
  • display: Formato de la fórmula (block o inline).
